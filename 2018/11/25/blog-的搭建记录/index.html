blog 的搭建记录~

发表于 2018-11-25 | 分类于 hexo | 评论数： | 阅读次数：

前言

  • 读书这么多年，一直坚信 “ 好记性不如烂笔头 ”
  • 大学时候喜欢纸质笔记，上了研究生后发现 Markdown 写笔记风格简约清晰，是个不错的文本记录工具，偶尔会用来写写汇报总结之类..
  • 在chrome 上搜索资料的时候，看到很多很有意思的 blog, 萌生了我想拥有有个用来记录自己学习生活的平台的想法，遂入坑~
  • 于是，在 2018.11.25 ，个人的 blog 上线啦~~

如何使用 hexo 搭建个人blog

对未来的展望

  • 慢慢觉得写作是个强大的技能，见过很多大佬技术不仅技术做得好，还能说会道，又写得一手好报告，着实羡慕.
  • 反观自己，常常发现脑子里想了很多，输出的文字总觉得缺了点火候.
  • 希望自己能利用好 blog 的平台，多学习，多总结，多输出~
  • 多年后回头看看自己的经历，也是个不错的回忆呢~
